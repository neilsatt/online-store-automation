CD into your project folder and run this command:

npm install --save-dev webdriverio selenium-standalone



Install stand alone selenium server:
node_modules/.bin/selenium-standalone install

Start the server:
node_modules/.bin/selenium-standalone start


Setting up the wdio config (use Power Shell)
run this command - a wdio folder should appear
	ls node_modules/.bin/
Then run this to set up/create your wdio.config file:
	./node_modules/.bin/wdio
